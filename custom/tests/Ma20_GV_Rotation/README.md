# Ma20_GV_Rotation — 测试基线

## 核心思想
成长/价值比价的 **20 日均线轮动** 策略：
- 比价 = 成长指数 / 价值指数
- 比价 > 20 日均线 → 持有成长
- 比价 < 20 日均线 → 持有价值
- 附加 DCD 方向冷却、DC 方向确认、BIAS 超买保护、E5 复燃冷却

## 来源
从研究阶段的 `X14_统一策略干净版` 通过蒙特卡洛检验后升入测试阶段。

## 研究阶段验证结果

| 检验 | 结果 | 说明 |
|:----|:---:|:-----|
| 块随机化 | ✅ 通过 | Sharpe 位于 100% 分位，不靠时序巧合 |
| 参数微扰 | ✅ 通过 | DCD/DC/BIAS/E5 参数稳定，无尖峰过拟合 |
| 信号随机化 | ✅ 通过 | 原始信号战胜 100% 随机信号 |
| 均线周期敏感性 | ✅ MA=20 最优 | 5~53 天扫描，MA=20 附近为稳定高原 |
