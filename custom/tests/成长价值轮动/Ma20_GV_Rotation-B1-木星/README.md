# B1-木星 — 成长价值轮动测试基线

## 代号说明
- **B1**: Testing phase Baseline 1
- **木星**: 第一颗被发现的基线，最大的行星

## 核心思想
成长/价值比价（RATIO）的 **20 日均线轮动**，辅以多层信号过滤：
1. **方向确认（DC=5）** — T值连续5天同号才确认方向
2. **方向冷却（DCD=6）** — 切换后锁定6天，防止反复跳仓
3. **弱信号空仓** — `|T|<1.3` 且 `|SLOPE|<0.002` 时空仓
4. **B2价值动量过滤** — 价值自身动量下跌时阻止买入（防价值陷阱）
5. **BIAS超买保护** — 价格偏离均线 >19% 时降仓 50%
6. **E5复燃冷却** — 20日跌幅 >9% 触发止损，冷却 8 天

## 来源
从研究阶段 `X14_统一策略干净版` 通过 Pipeline 2.0 全项检验后升入测试阶段。

| 参数 | 值 | 含义 |
|:----|:--:|:-----|
| MA | 20 | 比价均线周期 |
| DC | 5 | 方向确认天数 |
| DCD | 6 | 方向冷却天数 |
| rt | 1.3 | T弱信号阈值 |
| slope_thresh | 0.002 | 斜率弱信号阈值 |
| bias_high | 0.19 | BIAS超买阈值 |
| bias_mode | 'clear' | BIAS降仓50% |
| st | 0.09 | E5止损阈值(20日跌幅) |
| cd | 8 | E5冷却天数 |
| ms/ml | 10/20 | B2动量窗口 |
| bias_t_constraint | False | BIAS不加T约束 |
| e5_reset | True | E5重置冷却期 |
