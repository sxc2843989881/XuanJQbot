#### 步骤 3：编写因子插件
新建 `custom/reports/factor_plugins/<name>_plugin.py`：

```python
"""<Name> 因子插件"""
from typing import List, Optional
import pandas as pd

from custom.reports.contract import BacktestData
from custom.reports.factor_plugins.base import FactorPlugin, FactorPlotSpec


class <Name>FactorPlugin(FactorPlugin):
    """<Name> 因子插件"""
    
    def __init__(self, param1=default1, param2=default2):
        self.param1 = param1
        self.param2 = param2
    
    def plugin_name(self) -> str:
        return "<Name>"
    
    def prepare(self, data: BacktestData) -> BacktestData:
        """计算因子、信号、金叉死叉点"""
        # 1. 计算因子
        factor = calculate_factor(data.closes, self.param1)
        data.factors["<FactorName>"] = factor
        
        # 2. 计算信号
        signal = pd.Series(0, index=data.dates, dtype=int)
        signal[<buy_condition>] = 1
        signal[<sell_condition>] = -1
        data.signals = signal
        
        # 3. 计算金叉/死叉
        sig_diff = signal.diff().fillna(0)
        golden_cross = list(data.dates[sig_diff > 0])
        death_cross = list(data.dates[sig_diff < 0])
        data.cross_points = (golden_cross, death_cross)
        
        return data
    
    def factor_plot_specs(self) -> List[FactorPlotSpec]:
        return [
            FactorPlotSpec(
                name="<FactorName>",
                label="<FactorName>（因子）",
                color='#FF6B35',
                linewidth=1.3,
            ),
        ]
    
    def signal_yticklabels(self) -> Optional[List[str]]:
        return ['空头(-1)', '空仓(0)', '多头(+1)']
```

## 六、常见操作示例

### 6.1 从 baostock 获取数据
```python
import baostock as bs
import pandas as pd

def fetch_data(code, start_date, end_date):
    bs.login()
    rs = bs.query_history_k_data_plus(
        code,
        "date,open,high,low,close,volume",
        start_date=start_date,
        end_date=end_date,
        frequency="d",
        adjustflag="2",  # 前复权
    )
    data_list = []
    while rs.next():
        data_list.append(rs.get_row_data())
    df = pd.DataFrame(data_list, columns=rs.fields)
    df['date'] = pd.to_datetime(df['date'])
    df.set_index('date', inplace=True)
    for col in ['open', 'high', 'low', 'close', 'volume']:
        df[col] = pd.to_numeric(df[col])
    bs.logout()
    return df
```

## 八、常见问题

### Q1: 回测报错 "baostock 登录失败"
**原因**：网络问题或 baostock 服务不可用
**对策**：
1. 检查网络连接
2. 稍后重试
3. 使用本地缓存数据（`custom/data/etf_klines/`）

### Q2: matplotlib 中文乱码
**原因**：字体配置问题
**对策**：plotter.py 已配置 SimHei/Microsoft YaHei，若仍乱码：
```python
import matplotlib
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False
```

### Q3: 报告生成失败 "closes 长度 != dates 长度"
**原因**：数据契约不一致
**对策**：
```python
data.validate()  # 调用校验方法查看具体错误
```

### Q4: 如何添加新因子插件
1. 在 `custom/reports/factor_plugins/` 新建 `<name>_plugin.py`
2. 继承 `FactorPlugin` 抽象基类
3. 实现 `plugin_name()` / `prepare()` / `factor_plot_specs()`
4. 参考 `sma_plugin.py` 实现

## 九、最佳实践

### 9.1 命名规范
- 研究脚本：`research_<name>.py`
- 策略文件：`<name>_strategy.py`
- 因子文件：`factor_<name>.py`
- 回测脚本：`backtest_<name>.py`
- 测试脚本：`test_<module>.py`

### 9.2 文档规范
- 每个文件顶部写 docstring，说明用途
- 关键函数写 docstring，说明参数和返回值
- 中文注释，量化术语保留英文（Sharpe、Calmar、IC 等）

### 9.3 数据规范
- 默认从网络获取最新数据
- 本地缓存仅作离线备用
- 前复权数据（adjustflag="2"）

### 9.4 风控规范
- 回撤定义：完整周期（不按年切分）
- 必须包含交易成本（commission ≥ 0.001）
- 必须包含滑点（建议 0.0003）

### 9.5 报告规范
- 使用通用 reports 模块生成报告
- 不要在每个策略里重写绘图代码
- 新策略只需写 FactorPlugin 插件
