### 1.3 V72 的 MA75 实际逻辑（代码确认）

来源：[two_etf_rotation_v72.py:99-138](file:///C:/XuanJLH/Qbot/custom/research/成长价值轮动/v72/项目代码/two_etf_rotation_v72.py#L99-L138)

# 因子3：MA75择时（系统性下跌降仓）
g_ma = g_close.shift(1).rolling(MA_W).mean()   # MA_W=75
v_ma = v_close.shift(1).rolling(MA_W).mean()
both_below = (g_close.shift(1) < g_ma * MA_PCT) & (v_close.shift(1) < v_ma * MA_PCT)
# MA_PCT=0.97 → 跌破MA75的3%=2.3σ

# 状态机：MA75择时优先
if row["both_below"]:
    position.iloc[i] = CUT  # CUT=0.1 → 降仓到10%

```

**关键事实**：MA75 不是"因子"，而是"择时降仓位规则"。触发条件是"成长100 与 价值100 同时跌破各自 MA75 的 97%"，即**系统性下跌**（双指数齐跌）才降仓。触发后总仓位降至 10%（成长价值各半 + 90% 现金）。

## 二、三方观点：退化根因诊断

### 2.1 A1 研究员（Alpha 思想派 · 动量趋势组）

> **核心判断**：V72→V75 的退化不是"方向错了"，而是"减法砍错了对象"——砍掉了有效的趋势防御（MA75），又加了无效的噪声过滤（3日门控）。MA75 是趋势跟随派的核心盟友，绝非无用逻辑。

**对四个退化的根因判断**：

1. **V72→V74 退化（去 MA75）**：MA75 是"趋势择时防御层"。去掉它，系统性下跌时策略从"降仓硬扛"退化为"满仓硬扛"。数据为证：去 MA75 后年化 -1.76pp、回撤 +5.3pp。这 5.3pp 的回撤恶化不是偶然，而是**趋势防御缺失的必然代价**。知识库证据：[成长价值风格轮动研究综述.md](file:///C:/XuanJLH/玄机量化/02_行业轮动/03_成长价值风格轮动研究综述.md) 第一节第 4 条明确写"风控是收益主要来源：纯动量在熊市/震荡市高位接盘放大回撤。主流防御：大盘趋势过滤(MA200)、绝对动量择时、RSRS择时"——MA75 正是"大盘趋势过滤"的本地实现。

2. **V73 彻底失败（绝对动量主驱动）**：绝对动量是"确认型滞后指标"，二元全仓/全 cash 切换在震荡市必然 whipsaw。知识库证据：[动量因子优化与撤退信号研究.md](file:///C:/XuanJLH/玄机量化/02_行业轮动/04_动量因子优化与撤退信号研究.md) 2.5 节 Dual Momentum 实盘复盘——"2015-2024 实盘衰减：单一 12 个月 lookback 在震荡市 whipsaw；二元决策（全仓/全 cash）导致反弹错过"。V73 的 cash 占比 32.8% 正是"二元决策错过反弹"的本地复现，年化崩到 15.42% 是 whipsaw 的代价。

3. **V74→V75 退化（3日门控）**：已证伪（见 [researcher_review.md](file:///C:/XuanJLH/Qbot/custom/research/成长价值轮动/v75/回测结果/researcher_review.md)）。3 日纯动量违背 KST 多周期降噪原则（Martin Pring 1992，噪音降 62%），是"用噪声过滤降噪信号"的逻辑倒置。

4. **整体趋势诊断**：演进犯了"该砍的没砍、不该砍的砍了"的方向性错误。用户"因子要少"原则是对的（知识库：帆船 2 因子极简 V56 达 31.34% 新最优，2-3 因子极简优于 7+ 层堆叠），但 **MA75 不是因子，是择时层，不该当因子砍掉**。

**回退建议**：回 V72。理由：V72 年化 39.35% > V74 37.59%（+1.76pp），回撤 -48% < V74 -53.30%（好 5.3pp），Calmar 0.82 > V74 0.705（+0.115）。V72 在收益与回撤上**双杀** V74，唯一略逊的是 Sharpe（1.36 vs 1.372，差 0.012 可忽略）。

### 2.2 A2 研究员（Alpha 思想派 · 风格波动组）

> **核心判断**：MA75 降仓的"保护对象"是系统性下跌（both_below：成长价值齐跌破），这正是回撤恶化的来源。用户"比价均线过滤噪声优于纯动量"的洞察完全正确——V73 失败反向验证了它。

**对四个退化的根因判断**：

1. **V72→V74 退化**：MA75 的 `both_below` 条件精准命中"系统性下跌"——只有成长价值**同时**跌破 MA75 才降仓，单边下跌不降仓。这意味着它只在"无风格可逃"的普跌市降仓，不影响"分化期选强者"的 alpha。去掉它，等于在系统性下跌时放弃防御、满仓接刀。5.3pp 回撤恶化全部来自这部分。**用户洞察验证**：[失败分析.md](file:///C:/XuanJLH/玄机量化/06_失败教训与避坑/01_价值成长风格轮动四策略失败分析.md) 2.3 节帆船失败点 2——"流动性信号仅在极端情况触发，大多数熊市中帆船仍然硬扛，空仓占比仅 0.4%，有保护机制几乎是装饰"。MA75 正是帆船缺失的那个"真正会触发的熊市防御"。

2. **V73 彻底失败**：V73 用绝对动量作主驱动，cash 占比 32.8%，违反 A2 的"机会成本红线：cash 占比不得超过 15%"。绝对动量在分化期（一涨一跌）会错误地把正动量者也切到 cash，错过 alpha。用户洞察"比价均线过滤噪声优于纯动量"在此得到完美验证——**比价 MA20 是相对动量（选风格），噪声小；绝对动量是时序动量（择时），噪声大误判多**。

3. **V74→V75 退化**：3 日动量门控与 f2（动量加速度）同属动量族，存在共线性风险（知识库反模式 3：[失败分析.md](file:///C:/XuanJLH/玄机量化/06_失败教训与避坑/01_价值成长风格轮动四策略失败分析.md) 4.3 节"因子共线性被忽略"）。门控"保护成长"的初心对，但用错信号——应该用波动率预测或相对强度，而非 3 日纯动量。

4. **整体趋势诊断**：演进方向是"减法"，但减法对象错了。用户"因子要少"应理解为"信号因子要少"（如帆船 3 信号、玄铁已嫌 11 因子过多），而**择时降仓是仓位层不是因子层，不在"少因子"的射程内**。

**回退建议**：回 V72。但需修正 MA75 的执行——"降到 10%"一刀切过于激进，在牛市偶尔误降仓会损失收益（这正是 V72 年化被 V74 追近的部分原因，尽管 V72 仍高 1.76pp）。建议：MA75 触发后降仓到 30-50% 而非 10%，保留防御同时减少牛市误杀。

### 2.3 B 研究员（风控派 · 夏普回撤组）

> **核心判断**：从风控指标看，V72 全面优于 V74——Sharpe 几乎相同（差 0.012 可忽略），但回撤好 5.3pp、Calmar 高 0.115。MA75 是有效风控，但"降到 10%"是粗糙的一刀切，应升级为分档降仓。

**对四个退化的根因判断**：

1. **V72→V74 退化**：B 的硬指标对比——

   | 指标 | V72 | V74 | 风控评价 |
   |------|:---:|:---:|---------|
   | Sharpe | 1.36 | 1.372 | 几乎相同（+0.012 噪声级）|
   | 回撤 | -48% | -53.30% | **V72 好 5.3pp** |
   | Calmar | 0.82 | 0.705 | **V72 高 0.115** |

   5.3pp 的回撤改善是实打实的。知识库铁证：[失败分析.md](file:///C:/XuanJLH/玄机量化/06_失败教训与避坑/01_价值成长风格轮动四策略失败分析.md) 3.2 节"真正的飞跃来自防御能力"——玄铁回撤从 V1 的 -41.4% 压到 -17.46%（改善 24pp），"少亏比多赚更重要——亏 50% 需要赚 100% 才能回本"。MA75 提供的 5.3pp 回撤改善，正是"少亏"的价值。

2. **V73 彻底失败**：cash 占比 32.8% 说明空仓过多，是"机会成本黑洞"。从风控看，V73 回撤 -51.05% 比 V72 -48% 还差，说明绝对动量择时不仅没控住回撤，反而因 whipsaw 放大了回撤。这证明"用滞后指标择时"比"不择时"更糟。

3. **V74→V75 退化**：门控对 Sharpe/回撤/Calmar 三维度零贡献（全部持平或下降），是"装饰性信号"的又一例（知识库反模式 4：[失败分析.md](file:///C:/XuanJLH/玄机量化/06_失败教训与避坑/01_价值成长风格轮动四策略失败分析.md) 4.4 节"装饰性信号：触发率<5% 需重新评估价值"——V75 门控触发率 2.15% 接近装饰性边界）。

4. **整体趋势诊断**：演进在风控维度是单调退化的——V72(-48%) → V74(-53.30%) → V75(-53.30%)，回撤一路恶化或持平。原因是连续两次"去防御"（去 MA75）和"加噪声"（3 日门控），都指向风控削弱。

**回退建议**：回 V72。但 MA75"降到 10%"是粗糙的一刀切，B 建议升级为**分档降仓**（参考玄铁 7 熊市因子 3/7 投票的分档思路，但避免其共线性陷阱）：浅跌破（97%）降到 50%，深跌破（94%）降到 20%。这样既保留系统性下跌防御，又减少牛市误杀。

**联合修正**：回 V72 + MA75 分档降仓（50%→20%）。这是 2v1 共识的进攻核心（保留 MA75 趋势防御）+ 不压收益的风控补充（分档替代一刀切）。

### 4.3 实施步骤

1. **回退至 V72 代码**（恢复 MA75 择时逻辑），删除 V73/V74/V75 的偏离改造。
2. **MA75 分档降仓改造**（唯一改动点，不压收益）：

   if row["both_below"] and both_deep:
       position.iloc[i] = 0.2   # 深跌破降到 20%
   elif row["both_below"]:
       position.iloc[i] = 0.5   # 浅跌破降到 50%
   ```
   理由：浅跌破保留半仓（减少牛市误杀，提升收益），深跌破重仓撤离（控回撤）。这是"让收益更可持续"的服务性风控，不改变"系统性下跌必降仓"核心。
3. **V73/V74/V75 代码归档**为"已验证退化路径"备查，避免重蹈覆辙。

### 4.4 预期效果

- 年化收益：维持或略升（分档减少牛市误杀，预计 39-40%+，不低于 V72 基准 39.35%）
- 回撤：维持 -48% 或略改善（深跌破重仓撤离更果断）
- Calmar：维持 0.82+
- **不压收益硬约束满足**：分档是"牛市少误杀+熊市更果断"，双向有利

### 方向 1：RSRS 领先撤退补充 MA75 滞后性（A1+B 联合推荐）

### 方向 2：KST 多周期动量替代 f2 单周期加速度（A1 推荐）

## 六、整体趋势诊断总结

### 6.1 退化根因一句话

> **V72→V75 的退化是"减法砍错了对象"**：砍掉了有效的趋势防御（MA75），又加了无效的噪声过滤（3日门控），中途还试过被证伪的绝对动量主驱动（V73）。用户"因子要少"原则正确，但误把"择时层"当"因子层"砍掉。

### 6.2 退化的系统性问题诊断

| 问题 | 判断 | 说明 |
|------|:----:|------|
| 方向错了吗？ | **否** | "做减法"方向正确（知识库：帆船 3 信号反超七星 7 信号）|
| 执行错了吗？ | **是** | 减法对象错误（砍 MA75 择时层）+ 加法对象错误（加 3 日门控噪声）|
| 是参数拟合吗？ | 否 | V72 参数基于逻辑（MA75=3.5月趋势，0.97=2.3σ），退化是结构改动非调参 |
| 用户洞察错了吗？ | **部分** | "比价优于动量"正确（V73 反证）；"MA75 无用"错误（V74 反证）|

### 6.3 关键判断回答

| 关键问题 | 明确回答 |
|---------|---------|
| MA75 降仓到 10% 是"无用逻辑"还是"有效风控"？ | **有效但粗糙的风控**——5.3pp 回撤改善是实证，但一刀切 10% 应升级分档 |
| 用户的"因子要少"原则与 MA75 冲突吗？ | **不冲突**——MA75 是择时层不是因子层，不在"少因子"射程内 |
| 如果只能选一个版本，选 V72 还是 V74？ | **V72**——收益(+1.76pp)+回撤(好5.3pp)+Calmar(+0.115)三杀，仅 Sharpe 微逊可忽略 |
