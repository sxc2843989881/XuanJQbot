## 2. 核心创新

### T指标统一（关键突破）
将所有信号统一到单一 T 指标：
```
T = RATIO_DEV_Z = (RATIO - RATIO_MA20) / RATIO_DEV_STD20
```
- T > 0 → BULL（growth方向），T < 0 → BEAR（value方向）
- |T| < rt 且 |斜率| < slope_thresh → both_weak → 空仓
- 消除了 X61 中 F1（比价vs MA20）+ F0（z-score偏离度）的信号冗余

### 两个"完全空仓"突破

| 突破 | 版本 | 逻辑 | 效果 |
|------|------|------|------|
| **Br=0** | v25 U151 | G/V_BIAS > 0.19 时直接清仓（而非减仓） | Calmar +0.008 |
| **MHr=0** | v26 U164 | 持仓 ≥ 92天时直接清仓（而非减仓） | Calmar +0.014 |

### 其他关键优化
- **B2双周期**: V_MOM10 ≤ 0 **AND** V_MOM20 ≤ 0 → 改growth（比X61的单周期更精确）
- **E5激进止损**: st=0.088（8.8%跌幅）、sw=0.17/0.16、cd=8/9天
- **方向冷却**: dcd=6天，方向切换后6天内不再切换

## 3. 策略逻辑（9层）

```
第1层 F1-基础信号:     T=RATIO_DEV_Z>0 → BULL(growth), T<0 → BEAR(value)
第2层 A1-方向确认:     连续dc=5天F1信号一致才确认方向
第3层 方向冷却期:       dcd=6天, 方向切换后6天内不再切换
第4层 T+斜率双重确认:   |T|<rt=1.3 且 |斜率|<0.002 → both_weak → 空仓
第5层 B2-价值动量过滤:  value方向且V_MOM10<=0 AND V_MOM20<=0 → 改growth
第6层 BIAS过滤(Br=0):  G_BIAS>0.19或V_BIAS>0.19 → 完全空仓
第7层 E5-止损降仓:      20日跌幅>8.8% → 降仓至sw(17%或16%)
第8层 E5冷却期:         cd=8或9天, 降仓后冷却期内不恢复
第9层 持仓时间限制:     持仓>=92天 → 完全空仓(MHr=0)
```

## 5. 框架极限确认

经过29轮深度优化（v1-v29），已确认当前框架极限：

| 指标 | 极限值 | 达到轮次 |
|------|--------|---------|
| 无滑点 Calmar | **2.378** | v26 U164 |
| 滑点 Calmar | **2.059** | v27 U175 |
| 年化收益 | **49.79%** | v26 U164 |
| 最大回撤 | **-20.94%** | v26 U164 |

**验证无效的方向（8个全部失败）**：
1. 波动率过滤 → 交易次数暴增
2. RSI过滤 → 年化大降至35-47%
3. 累计涨幅过滤 → 年化下降
4. 双BIAS过滤 → 长周期阈值无效
5. 信号平滑 → smooth_days 2-20 无改善
6. 双重止损 → st2/sw2组合无改善
7. 趋势强度过滤 → TS参数无改善
8. BIAS恢复确认 → recovery_days无改善

**结论**: 需新数据源或新因子系统才能进一步突破。
