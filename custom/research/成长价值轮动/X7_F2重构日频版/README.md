## 设计要点

### F1保留（不变）
```python
ratio = (g_close / v_close).shift(1)
ratio_ma20 = ratio.rolling(20).mean()
ratio_dev = ratio / ratio_ma20 - 1
f1_signal = np.tanh(ratio_dev * 30) * 0.5
```

### F2'重构（比价rolling Sharpe）
```python
ratio_ret = ratio.pct_change()
ratio_mom = ratio_ret.rolling(20).sum()       # 20日比价累计收益
ratio_vol = ratio_ret.rolling(20).std()       # 20日比价波动率
ratio_sharpe = ratio_mom / (ratio_vol * np.sqrt(20))
f2_signal = np.tanh(ratio_sharpe * 1.0) * 0.3
```

2. **X7-1虽微超X2但回撤恶化**：+0.17pp收益不抵+5.54pp回撤

4. **F2'零回撤贡献**：F2'恶化回撤0.54pp，而F2原版改善回撤5pp
