## 三、优化内容

### 3.1 F2 因子分析（保留原版）

X2 Task 1 完成了 F2 因子场景有效性分析，结论：**F2 在 X1 中的用法已接近最优，三个优化方案均未超越 X1**。

**分析结果：**
| 场景 | 频率 | F2 准确率 | 决策变化率 | 结论 |
|:----:|:----:|:---------:|:----------:|:----:|
| 成长初期启动 | 18.25% | 45.60% | 23.20% | 有害噪声（准确率<50%但频繁改决策） |
| 成长末期减速 | 3.50% | 52.17% | 0.00% | 对 candidate_g 判定无效 |
| 趋势中段 | 17.81% | 73.95% | 0.00% | 对但冗余（与 F1 同向，不改变决策） |
| 其他 | 60.44% | 56.52% | 6.52% | 有正预测力，是 F2 实际起作用的主要场景 |

**优化尝试（均未超越 X1，故 X2 保留 F2 原版）：**
- F2 同向强化信号用于仓位调整（趋势中段加仓）
- F2 反向负信号用于风控减仓（成长末期减速降成长敞口）
- 在"成长初期启动"场景降低 F2 权重或加反向过滤
- 放宽 clip 阈值（accel_diff 频繁触顶 ±2.000%）

详见 `回测结果/f2_analysis_report.txt` 和 `项目代码/test_f2_optimization.py`。

### 3.2 降仓逻辑优化（方案A 分段降仓）

**X1 降仓逻辑（基线）：**
- MA75 双跌破（close < MA75 × 0.97）→ 降仓到 10%（成长5% + 价值5%）

**X2 正式版降仓逻辑（方案A 分段降仓）：**
```
每周五决策：
  ├─ 若 MA75 双跌破（close < MA75×0.97）
  │     → 降仓到 25%（在当前方向上）
  ├─ 若 MA50 双跌破（close < MA50×0.97）
  │     → 降仓到 50%（在当前方向上）
  └─ 否则 → 100% 仓位（按 style_score 方向：100%成长 或 100%价值）

执行：T日信号 → T+1日开盘价执行
频率：周频调仓（W-FRI，约15次/年）
```

**分段降仓的优势：**
1. MA50 提前预警：在 MA75 触发前先减半仓，规避部分下跌
2. MA75 进一步降仓：跌势确认后再降到 25%，但保留部分敞口避免踏空反弹
3. 与 X1 的二元开关相比，对市场状态响应更精细

**对比方案（保留代码用于参考，未采用为正式版）：**
- 方案 B（直接 0%）：MA75 双跌破即清仓。Calmar 最优但年化低于 X1，违反"收益为先"
- 方案 C（三档降仓）：MA50→50%, MA75浅→25%, MA75深→0%。综合介于 A、B 之间

### 3.3 实现瑕疵修复

**X1 的实现瑕疵：**
X1 在降仓时 `direction` 硬编码为 `'value'`，与当时 style_score 的实际方向可能不一致。这是已知实现瑕疵，会影响降仓期间的收益归因。

**X2 的修复：**
X2 所有方案（A/B/C）在降仓时 `direction` 由 `style_score > 0` 决定：
- `style_score > 0` → `'growth'`
- `style_score <= 0` → `'value'`

即使权重降到 0.25（方案A）或 0.0（方案B），方向仍跟随因子信号，确保降仓期间的方向选择符合当前因子判断。

## 四、量化研究员评价结论

### 4.1 2v1 评价核心结论

**评价形式：** 2 位 Alpha 派研究员（追求最高收益）vs 1 位风控派研究员（控制夏普/回撤）

**核心结论：**
1. **X2 正式版应采用方案 A（分段降仓）**，而非 Task 5 默认选择的方案 B
2. **理由：** 方案 B 年化 36.14% 低于 X1 基准 36.29%，违反"收益为先"原则
3. **方案 A 在四项核心指标上均小幅优于 X1**，符合 Alpha 派主张
4. 风控派保留意见：方案 B 在 Calmar/回撤上更优，但收益劣化不可接受

### 4.2 X3 优化方向建议

| 优先级 | 建议 | 说明 |
|:------:|:----:|:----:|
| P0 | 降仓改回方案 A | ✅ 已完成（X2 采用方案 A） |
| P1 | F1 升级为 KST 多周期动量 | 替换比价 MA20 方向，捕捉多周期动量 |
| P1 | style_score 连续加权 | 都涨超配成长（不再二元全仓切换） |
| P2 | F2 加 ADX 过滤 | 强趋势市场才相信 F2，弱趋势过滤噪声 |
| P2 | F2 clip 放宽测试 | 当前 clip(-0.02, 0.02) 频繁饱和 |
| P3 | 加 RSRS 领先撤退 | 时变阻力支撑，领先 MA 退出（前提不压收益） |

## 九、已知问题 & X3 改进方向

### 已知问题
1. **改进幅度有限**：方案 A 相对 X1 各项指标仅小幅提升（年化 +0.05pp），优化的边际收益递减
2. **调仓次数增加**：+16 次（183→199），因 MA50 提前降仓引入额外调仓
3. **F2 价值有限**：场景分析显示 F2 在 60% 的"其他"场景才有正预测力，且在"成长初期启动"是有害噪声
4. **降仓仍偏粗糙**：MA50/MA75 阈值固定，未考虑波动率自适应

### X3 优化方向（来自量化研究员评价）
- P1：F1 升级 KST 多周期动量 + style_score 连续加权（都涨超配成长）
- P2：F2 加 ADX 过滤 + clip 放宽测试
- P3：加 RSRS 领先撤退（前提不压收益）
