## 一、版本定位

X1 是成长100/价值100轮动策略的**研究阶段稳定基准版**。
所有后续优化版本（Y-series / V-series）均需与此版本对比，以量化改进幅度。

**选择 V72 作为基准的原因：**
1. 因子精简，逻辑透明，无过度拟合风险
2. 参数基于经济学逻辑选择（非搜索拟合）
3. 经过多个版本迭代验证，稳定性高
4. 新轻量化引擎已验证，结果可复现

## 二、策略逻辑

### 因子层（2个风格因子 + 1个择时因子）

| 因子 | 权重 | 说明 |
|:----:|:----:|------|
| **f1 比价MA20方向** | 0.5 | 成长/价值比价偏离20日均线的幅度，tanh(ratio_dev×30)压缩到±0.5 |
| **f2 动量加速度** | 5.0 | 21日ROC的10日变化率，clip(-0.02, 0.02)防过度反应 |
| **MA75择时** | — | 两者均跌破75日均线×0.97 → 系统性风险 → 降仓到10% |

### 仓位层

```
每周五决策：
  ├─ 若 MA75 双跌破 → 降仓10%（成长5% + 价值5%）
  └─ 否则 → 按 style_score 方向：100%成长 或 100%价值

执行：T日信号 → T+1日开盘价执行
频率：周频调仓（约9.6次/年）
```

## 六、已知问题 & 改进方向

### 已知问题
1. **回撤偏大**（-37.69%）：MA75择时是二元开关，过于粗糙，熊市初期反应慢
2. **都涨时没超配成长**：二元全仓切换错过alpha机会
3. **印花税缺失**：指数无印花税（合理），但若用于ETF需加万3
4. **降仓处理简化**：当前10%在单方向，原版是各持5%

### 优化方向（待验证）
1. MA75分档降仓（浅跌50%/深跌20%）
2. 四状态判断 + 都涨超配成长
3. RSRS领先撤退指标
4. 双跌阴跌兜底机制
