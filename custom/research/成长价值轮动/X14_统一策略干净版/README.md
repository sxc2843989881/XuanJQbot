# X14 — 统一策略干净版

## 与原版 X13 的区别

X14 是在 X13(v26 U164, Calmar 2.378)基础上，去掉了所有不合逻辑的参数后的合理化版本：

| 改动 | 原版 X13 | X14 干净版 | 理由 |
|------|---------|-----------|------|
| 最大持仓天数 | max_hold_days=92, max_hold_reduce=0.0 | **已移除** | 固定日历天数无任何金融逻辑支撑 |
| E5止损阈值 | st=0.088 | **st=0.09** | 取整，避免精度过拟合 |
| BIAS降仓比例 | bias_reduce=0.0 | **bias_mode='clear'(清仓)** | 改为用户可选的模式参数 |

## 表现

| 指标 | X14 clear模式 | X14 ignore模式 | X13 原版 |
|------|:-----------:|:-------------:|:-------:|
| 年化 | 45.27% | 42.87% | **49.40%** |
| 最大回撤 | **-23.20%** | -23.20% | -22.55% |
| Sharpe | 1.544 | 1.469 | **1.681** |
| Calmar | **1.951** | 1.781 | 2.191 |
| 交易次数 | 397 | 387 | 392 |
| 滑点(手续费1bps+冲击5bps) | **含** | **含** | 5bps |

X14 所有指标已含滑点（手续费1bps + 冲击5bps）。

## 6层逻辑

1. **方向确认(dc=5)** — T>0连续5天→成长，T<0连续5天→价值
2. **方向冷却(dcd=6)** — 切换后6天内不让再切
3. **T+斜率双重确认→空仓** — \|T\|<1.3 AND \|斜率\|<0.002 时空仓
4. **B2价值动量过滤** — 方向BEAR时，如果价值短/长周期都在跌，反手买成长
5. **BIAS超买清仓(bias_mode)** — 当前价偏离20日均线>19%时清仓；用户可选忽略
6. **E5止损** — 20日跌幅>9%时降仓到17%，冷却8天后恢复

## 参数

| 参数 | 默认值 | 含义 |
|------|--------|------|
| dc | 5 | 方向确认天数 |
| dcd | 6 | 方向冷却天数 |
| rt | 1.3 | T弱阈值 |
| slope_thresh | 0.002 | 斜率弱阈值 |
| ms | 10 | B2短周期 |
| ml | 20 | B2长周期 |
| bias_ma | 20 | BIAS均线周期 |
| bias_high | 0.19 | BIAS超阈值 |
| bias_mode | 'clear' | BIAS模式: clear(清仓)/ignore(忽略) |
| st | 0.09 | E5止损阈值 |
| sw | 0.17 | E5降仓比例 |
| cd | 8 | E5冷却天数 |

## 参数来源说明

各参数的制定依据分为三类：

### 基于量化金融学原理

| 参数 | 值 | 依据 |
|------|------|------|
| **dc (方向确认)** | 5天 | 避免单日毛刺造成频繁切换，具体天数属工程选择(3~7天均可) |
| **dcd (方向冷却)** | 6天 | 切换后锁定方向防止反复跳仓，具体天数属工程选择(3~10天均可) |
| **ms/ml (B2周期)** | 10/20天 | 短周期≈2周、长周期≈1月，行业通用窗口 |
| **bias_ma (BIAS均线)** | 20天 | 20日均线，技术分析行业标准 |
| **sw (E5降仓比例)** | 0.17(≈1/6) | 风控设计：快速砍仓到微量持仓观察，非优化结果 |

### 基于历史数据网格搜索

| 参数 | 值 | 说明 |
|------|------|------|
| **st (E5止损阈值)** | 0.09 | 搜索范围5%~15%，7~9%区间Calmar稳定在1.95±0.01，**未来可能随市场波动率漂移** |
| **rt (T弱阈值)** | 1.3 | 搜索范围0.5~3.0，1.3附近最优 |
| **slope_thresh (斜率弱阈值)** | 0.002 | 搜索范围0.0005~0.01，0.002附近最优 |
| **bias_high (BIAS超买阈值)** | 0.19 | 搜索范围0.10~0.30，0.19附近最优 |
| **cd (E5冷却天数)** | 8 | 搜索范围0~30，6~8天Calmar相近，8天降仓天数略多但回撤控制略好 |

### 用户自定义选项

| 参数 | 值 | 说明 |
|------|------|------|
| **bias_mode** | 'clear'/'ignore' | 取决于市场判断：极端行情开启('clear')，正常行情可忽略('ignore') |

### 使用建议

搜索优化的参数(st, rt, slope_thresh, bias_high, cd)在**历史数据上的最优值可能随市场环境变化**。建议：
- st=0.09 有±1%的容忍区间(7~9%效果相近)，不必频繁调整
- 如果市场波动率出现**系统性变化**(如创业板波动率从9%变为12%)，st应按比例缩放
