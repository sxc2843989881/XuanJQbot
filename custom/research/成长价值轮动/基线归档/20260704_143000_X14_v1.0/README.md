## 策略说明

X14 是成长价值轮动策略，标的为国证成长100R (480080) 和国证价值100R (480081)。

6层防守架构：
1. **第1层：方向确认 (dc=5)** — 连续5天T同号才确认方向
2. **第2层：方向冷却 (dcd=6)** — 方向切换后6天内不再切换
3. **第3层：T+斜率双重弱过滤** — T弱且斜率弱时空仓
4. **第4层：B2价值动量过滤 (ms=10, ml=20)** — 价值动量双负时反手成长
5. **第5层：BIAS超买清仓 (bias_ma=20, bias_high=0.19)** — 偏离超19%时清仓
6. **第6层：E5止损 (st=0.09, sw=0.17, cd=8)** — 20日跌超9%时降仓至17%

核心因子：`T = (RATIO - RATIO_MA20) / RATIO_STD20`，其中 `RATIO = G_CLOSE / V_CLOSE`
