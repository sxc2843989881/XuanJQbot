## 2. 策略逻辑（极简2因子）

```python
# 第1层：收益增强（F1 + A1 + B2）
ratio = g_close / v_close
ratio_ma20 = ratio.rolling(20).mean()
ratio_dev = ratio / ratio_ma20 - 1
f1_signal = np.tanh(ratio_dev * 30) * 0.5  # F1比价MA20方向

# A1: 斜率>0.3% + 4天方向一致确认
ma20_slope = (ratio_ma20 - ratio_ma20.shift(5)) / ratio_ma20.shift(5)
slope_ok = ma20_slope.abs() > 0.003
# 4天确认（X10是2天，X11-A改为4天）
dir_confirmed = base_dir.where(
    (base_dir == base_dir.shift(1)) &
    (base_dir == base_dir.shift(2)) &
    (base_dir == base_dir.shift(3)),
    np.nan
)
dir = dir_confirmed.where(slope_ok, np.nan).ffill()

# B2: value方向要求价值指数20日动量>0
wrong_value = (dir == 'value') & (v_mom20 <= 0)
dir[wrong_value] = 'growth'

# 第2层：风险防御（E5持仓止损）
# 持仓方向20日跌幅>10% → 降仓30%
growth_stop = (dir == 'growth') & (g_dd20 < -0.10)
value_stop = (dir == 'value') & (v_dd20 < -0.10)
weight = 1.0
weight[growth_stop | value_stop] = 0.3
```

**与X10的唯一差异**：A1确认从2天改为4天。这一个参数变化带来：
- 年化 +2.70pp（37.26% → 39.96%）
- Sharpe +0.069（1.238 → 1.307）
- 交易次数 -41次（261 → 220）
