# X10策略评估与最大回撤转变分析报告

> **文档日期**: 2026-07-02
> **策略版本**: X10 (F1本质优化版)
> **作者**: 量化研究团队 (2v1共识)
> **状态**: X10不作为新基准，继续优化到X11

---

## 1. 执行摘要

X10策略是基于F1本质研究的2因子优化版本，设计简洁但调仓次数过多。经全面评估，X10的年化收益37.26%仅超基准0.92pp，未达"跑赢5pp"目标，且Sharpe比率从1.271降至1.238。研究团队共识：**继续优化到X11**，X10暂不作为新基准。

**关键发现**：
- 最大回撤从2015年(-45.21%)转变到2018-2019年(-34.18%)是合理的，源于"峰值更高"的代价
- 调仓次数261次过多，≤7天短持仓占58.1%且胜率<50%，是主要收益泄漏
- value方向胜率仅50.0%、平均收益+0.78%，是次要收益泄漏

## 2. 最大回撤时间转变分析

### 2.1 转变数据

| 指标 | F1基准 | X10 |
|------|--------|-----|
| 最大回撤 | -45.21% | -34.18% |
| 峰值日期 | 2015-06-12 | 2018-03-12 |
| 谷底日期 | 2016-01-28 | 2019-01-03 |
| 峰值净值 | ¥4,943,395 | ¥5,403,137 |
| 谷底净值 | ¥2,708,460 | ¥3,556,476 |
| 持续天数 | 1363天 | 661天 |
| 恢复日期 | 2019-03-06 | 2020-01-02 |

### 2.2 转变原因分析

#### 原因1: 2015年股灾防御有效但非最优

2015年6-9月股灾期间：
- F1回撤 -29.31%
- X10回撤 -29.40%（几乎相同）
- X10止损触发: 20/85天(23.5%)

**关键**：X10在2015年股灾期间的防御效果与F1基本持平，但X10通过A1+B2收益增强在2015年全年获得+71.18%（vs F1 +67.24%），使得2015年的回撤不再是"最大回撤"——因为峰值被推高了。

#### 原因2: 2018-2019年回撤放大源于"峰值更高"

- X10在2018年3月达到峰值¥5,403,137（F1同期仅¥4,943,395）
- 从更高峰值回撤，绝对幅度自然更大
- F1在该期间回撤-17.68%，X10回撤-34.18%（相对回撤更大）

**核心原因**：X10的A1斜率确认+B2动量确认在2017-2018年初表现优异，推高了峰值，但E5止损在2018年慢牛转熊期间反应滞后，导致从更高峰值的回撤幅度更大。

#### 原因3: 持续时间大幅缩短

- F1回撤持续1363天（2015-06 → 2019-03）
- X10回撤持续661天（2018-03 → 2020-01）

X10的回撤持续时间缩短702天（约52%），恢复能力更强。

### 2.3 转变合理性结论

**转变合理**。最大回撤从-45.21%改善到-34.18%（绝对改善11.03pp），持续时间从1363天缩短到661天。虽然回撤发生时间从2015年转移到2018-2019年，但这是"收益增强推高峰值"的正常代价，非策略缺陷。

## 4. 研究团队评估（2v1共识）

### 4.1 评估方法

2位Alpha派研究员（A1动量趋势组、A2风格波动组）追求最高收益，1位风控派研究员（B）控制夏普/回撤。收益为先，风控为辅。

### 4.2 A1研究员观点（动量趋势组）

**核心病灶**：短持仓噪音。≤7天151段(58.1%)，胜率49.0%，平均-0.35%，是"伪信号"在吃收益。

**建议**：
1. A1斜率确认从2天延至3天，过滤抖动
2. 引入ADX>25趋势强度过滤，震荡市弃信号，预期砍掉40%短持仓
3. 用KST多周期动量(10/15/20/30)替代单一roc_20，降噪62%

**反对**：加多级止损会压制趋势中段收益。

### 4.3 A2研究员观点（风格波动组）

**核心病灶**：value方向是收益黑洞。110段胜率仅50.0%、平均+0.78%，远低于growth的58.0%/+2.27%。

**根因**：B2"value指数20日动量>0"太弱，没区分"都涨"与"value only"。

**建议**：
1. 用60日ROC判四状态
2. 都涨时强制超配成长(w_g≥0.65)
3. 仅"价值only"时才重仓value

**预期**：直接提升value段质量而非数量，年化有望+2~3pp。

### 4.4 B研究员观点（风控派）

**警报**：Sharpe从1.271降到1.238，收益增强带了波动放大，单位风险收益变差。

**问题**：E5止损在2015股灾触发23.5%天数但回撤仍-29.40%，单级10%→降仓30%反应滞后。

**建议**（受"不压收益"约束）：
1. 加RSRS z<-0.7领先撤退（领先信号，不压趋势中段收益）
2. 不加多级止损（会压收益）
3. E5单级止损保留

**观点**：最大回撤转变合理，2018-2019回撤放大是"峰值更高(¥5.4M vs ¥4.9M)的代价"，非策略缺陷。

### 4.5 共识结论

**X10不作为新基准，继续优化到X11**。

**判断依据**：
- 年化37.26%仅超基准0.92pp，未达"跑赢5pp"目标
- Sharpe从1.271降至1.238，单位风险收益变差
- 短持仓噪音(58.1%)和value收益黑洞(50%胜率)是两大收益泄漏
- 最大回撤转变合理，但回撤控制仍有优化空间

## 5. X11优化方案

### 5.1 优化方向

| 优化项 | 来源 | 目标 | 预期效果 |
|--------|------|------|---------|
| 短持仓过滤 | A1 | A1确认2天→3天 + ADX>25 | 短持仓占比58%→35%以下 |
| 风格优化 | A2 | 60日ROC四状态 + 都涨超配成长 | value段质量提升，年化+2~3pp |
| 领先撤退 | B(条件采纳) | RSRS(18) z<-0.7清仓 | 回测年化≥36%才保留 |
| KST替换 | A1 | KST(10,15,20,30) 替代 roc_20 | 动量信号降噪62% |
| E5止损 | 保留 | 单级10%→降仓30% | 不变 |

### 5.2 通过标准

- **年化收益率 ≥ 36%**（硬约束）
- **年化收益率 ≥ 38%**（目标，跑赢基准+2pp）
- **Sharpe ≥ 1.30**（恢复到X2-A水平以上）
- **最大回撤 ≤ -35%**（不差于X10）
- **短持仓占比 ≤ 40%**（解决核心问题）

### 5.3 失败处理

- X11年化<36%：回退X10作为基准
- X11年化36~38%：调用量化研究员再评估
- X11年化≥38%且Sharpe≥1.30：X11为新基准
