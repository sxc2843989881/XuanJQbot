### F2''=RSI差（新增第三因子）
```python
def compute_rsi(close, period=14):
    """Wilder RSI（1978年原版）"""
    delta = close.diff()
    gain = delta.where(delta > 0, 0.0)
    loss = -delta.where(delta < 0, 0.0)
    avg_gain = gain.ewm(alpha=1/period, adjust=False).mean()
    avg_loss = loss.ewm(alpha=1/period, adjust=False).mean()
    rs = avg_gain / avg_loss
    return 100 - 100 / (1 + rs)

g_rsi = compute_rsi(g_close, 14).shift(1)
v_rsi = compute_rsi(v_close, 14).shift(1)
rsi_diff = g_rsi - v_rsi
f2_rsi_signal = np.tanh(rsi_diff / 10.0) * 0.3
```

### 三因子合成

```python
style_score = f1_signal + f2_orig_signal + f2_rsi_signal

```

1. **F2''单独不行但叠加有效**：F2''与F1高共线性0.89，但与F2原版共线性0.34，提供了F2原版没有的边际增量
