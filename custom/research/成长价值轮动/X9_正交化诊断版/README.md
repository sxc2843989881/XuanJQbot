# X9 正交化诊断版（★ 最终最优解 / 优化循环第二次终止版）

> **版本类型**：最终最优解（基于X8-4做F2''-F1正交化诊断，确认F2''有真实独立alpha）
> **日期**：2026-07-02
> **性能**：年化 37.60% / 回撤 -37.66% / Sharpe 1.313 / **Calmar 0.999（历史首破1.0）**

## 1. 版本定位

X9 是对 X8-4（F2''=RSI差三因子叠加版）的因子独立性诊断。X8-4 虽然年化38.18%超X2-A 1.84pp，但F2''与F1 Pearson共线性高达0.89（违反<0.5约束），存在"一人投两票"嫌疑——F2''的+1.99pp边际增量中，有多少是F1重复计数？

X9 通过对F2''做F1正交化（OLS回归取残差），测算F2''去除F1线性影响后的独立alpha贡献。

## 2. 因子结构

| 因子 | 公式 | 权重 | 类型 |
|------|------|------|------|
| F1 | `tanh((ratio/ratio_ma20-1)*30) * 0.5` | 0.5 | 趋势类（比价MA20方向） |
| F2原版 | `accel_diff * 5.0` (clip±0.02) | 5.0 | 动量类（21日加速度差） |
| **F2''_perp** | `F2'' - (intercept + β×F1)` | 0.3 | **正交化反转类（RSI差残差）** |
| MA50/MA75 | 分段降仓 | - | 趋势防御（50%→25%） |

F2''_perp计算：
- 原始F2'' = `tanh((g_rsi14 - v_rsi14)/10) * 0.3`（Wilder RSI差）
- OLS回归：`F2'' = -0.0218 + 0.5589×F1 + ε`（R²=0.7987）
- 正交化残差：`F2''_perp = F2'' - (-0.0218 + 0.5589×F1)`
- **F2''_perp 与 F1 Pearson = -0.0000（完美正交）**
