# 安装指南

> 源文档：https://pmorissette.github.io/bt/install.html

## 安装 bt

### 通过 pip 安装（推荐）

```bash
pip install bt
```

由于 bt 有许多依赖，强烈建议先在 Windows 上安装 [Anaconda Scientific Python Distribution](https://store.continuum.io/cshop/anaconda/)，它预装了许多必要的包。

### 通过 conda 安装

```bash
conda install bt --channel conda-forge
```

### Python 版本要求

bt 兼容 Python >= 3.7。

## 推荐环境

建议使用 [IPython Notebook](http://ipython.org/notebook.html) 作为 bt 的开发环境。它可以：

- 在文档中内联显示图表
- 使用 Markdown 添加说明文字
- 轻松与同事共享 Notebook
- 保存为 PDF 格式
